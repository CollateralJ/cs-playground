# Unvigesimal Numbers!

Turns an unvigesimal string representation of a number into its decimal value and prints it. This project taught me how to use macros in x86.

The output is [here](https://github.com/CollateralJ/cs-playground/blob/main/x86/unvigesimal_macros/output.txt)

### Example:

number(_21) = bc8093i2

number(_10) = 20873925263

### Usage:

make 

./macros
