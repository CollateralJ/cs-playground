# Maze Traversal in x86

A project designed to learn the implementation details of functions in x86. There are two separate functions being tested: first an unrelated int2string function, and then the maze search algorithm. 

There is a sample output below, you can see the full output [here](https://github.com/CollateralJ/cs-playground/blob/main/x86/maze_search_int2string/output.txt)

![Solving Maze](https://github.com/CollateralJ/cs-playground/blob/main/x86/maze_search_int2string/mazesolve.png "solved!")

### Usage:

make 

./mazesearch
