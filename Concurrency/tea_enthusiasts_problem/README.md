# Tea Enthusiasts Problem

A producer-consumer problem project to learn safe practices for synchronization in multi-threaded programs. Locks and semaphores are used to accomplish this task.

The program creates 14 tea enthusiasts (consumers) and 7 suppliers (producers), each as their own thread, to work at the same time until each tea enthusiast has enjoyed 15-25 entire tea drinks!
