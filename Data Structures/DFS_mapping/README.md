# Map Navigation Using Depth First Search

Uses an adjacency list and node weights to navigate a simulated video game map (resident evil 3), and find a way to the courtyard safely.

Usage: 
./a.out (adjacency list: rpd.txt) (room weights: nemesis01.txt)

Outputs: Path to the exit, or death if there is no possible path.

Example output:

./a.out rpd.txt nemesis01.txt
